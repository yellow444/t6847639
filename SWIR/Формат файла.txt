Описание формата:
Кодировка - Little Endian
1) Заголовок
	1.1) Int - разрешение кадра по x
	1.2) Int - разрешение кадра по y
	1.3) Int - количество кадров в пакете n (для HDR)
	1.4) Double - Частота кадров
	1.5) Long - Адрес начала кадров
	1.6) Служебное место (1024 кБ)
	1.7) Таблицы калибровок (FS), Double размером x * y * n
	1.8) Таблицы калибровок (K), Double размером x * y * n
	1.9) Таблица дефектов, UInt размером x * y
2) Данные кадров
	Каждый кадр - последовательный набор значений (построчно), UShort размером x * y * n
